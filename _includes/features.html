Features

Quick & Easy Setup

Signup with Snipcart and change API key in the theme's configuration file.

Host on Static Servers

Host Dragon Fruit Jekyll e-commerce website on static servers like GitHub, Gitlab, Bitbucket etc.

Add Products with Ease

Add products in one file in the theme, it will show up in the product list.

Achieve Your Targets

Set snipcart API, change a few details, list your products and you are set to make sales!
